# 虚拟直播运营后台复刻版

这是一个与旧版 `robot-live-admin-prototype` 隔离的静态原型。

- 入口文件：`index.html`
- 样式文件：`styles.css`
- 交互文件：`app.js`
- 标注数据：`handoff-annotations.json`

后续维护方向：

- 当前文件用于静态原型查看，不代表最终工程目录。
- 后续如果改造代码框架，可以采用 Vue / Vite；每个左侧菜单独立维护为一个 `.vue` 页面组件。
- 公共表格、弹层、页签、资源选择器和标注工具可以抽成共享组件，避免菜单之间复制代码。

覆盖页面：

- 虚拟直播商品
- 管理直播间
- 机器人管理
- 技能管理
- 演出管理

主要交互：

- 左侧菜单切换
- 顶部标签页切换
- 创建机器人：角色设定 / 设备管理
- 音色情绪和大模型配置
- 百炼智能体下拉绑定
- 创建虚拟带货直播间
- 新增技能、关键词回车添加
- 新增演出、多条表演单元、排序和结束点配置
- 技能 / 演出批量下发
